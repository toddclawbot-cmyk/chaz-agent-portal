The 6-Step Reasoning Chain

Every question triggers a transparent, sequential reasoning process. Watch the agent think — from understanding the question to taking autonomous action.

STEP 01 INPUT
UNDERSTAND
Groq LLM (llama-3.3-70b-versatile) interprets the natural language question, identifies the intent, and determines what data and actions are needed.
Groq LLM Intent Classification Parameter Extraction
STEP 02 DATA
QUERY DATABRICKS
The LLM generates precise SQL from natural language, executes it against chazbakedgoods.sales — the bakery supply chain data warehouse.
NL → SQL Databricks SQL 9 Tables
STEP 03 ANALYSIS
ANALYZE RESULTS
Correlates inventory levels with 3-week demand forecasts. Identifies coverage gaps, depletion timelines, and supply constraints.
Demand vs Supply Coverage Ratio Trend Analysis
STEP 04 CRM
CROSS-REFERENCE SALESFORCE
Checks Salesforce for active opportunities, recent cases, and account health at any supplier or franchise locations flagged as at-risk.
Account Object Opportunity Object Case Object
STEP 05 DECISION
ASSESS RISK
Calculates business impact using coverage ratios, depletion velocity, and opportunity value at risk. Assigns a risk tier: Low / Medium / High / Critical.
Risk Scoring Business Impact Threshold Logic
STEP 06 ACTION
ACT
If risk threshold is met: creates a Salesforce Case with full context, drafts an outreach email via Apple Mail, and streams the result back to the user in real time.
Create Case Draft Email SSE Response

Core Capabilities

What the Agent Can Do

Six core capabilities that combine to create a production-grade AI agent capable of autonomous decision-making across multiple data systems.

Natural Language to SQL
Ask supply chain questions in plain English. Groq's llama-3.3-70b-versatile generates precise, syntactically correct SQL against the Databricks warehouse — no query language required.
Real-Time Event Streaming
Every step of the agent's reasoning is streamed back to the UI via Server-Sent Events (SSE). Watch the thought process unfold in real time, with full transparency into what the agent is doing.
Multi-System Orchestration
Coordinates reads and writes across Databricks and Salesforce simultaneously. Joins supply chain data with CRM context to build a complete picture of business risk.
Autonomous Action
When risk thresholds are exceeded, the agent acts — creating Salesforce Cases with full context and drafting outreach emails. No human intervention required for routine escalation.
Visible Reasoning Chains
The agent's 6-step reasoning trace is fully visible. Every decision is explainable — Understand, Query, Analyze, Cross-Reference, Assess, Act. No black boxes.
Risk Detection & Alerting
Configurable risk thresholds based on coverage ratios, depletion velocity, and business impact. Risk tiers (Low / Medium / High / Critical) trigger different escalation paths automatically.
